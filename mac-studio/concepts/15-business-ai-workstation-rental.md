# Business Concept 15: AI Workstation Rental Service

## The Business
Rent pre-configured Mac Studios with AI tools to creatives, developers, and businesses.

## Hardware
- **Multiple Mac Studios** (M1 Max/M2 Ultra)
- **Rack mount enclosures** (Mac Studio fits in 2U)
- **Network infrastructure**: 10GbE, VPN

## Revenue Streams
1. **Hourly rental**: €5-20/hour
2. **Daily rental**: €50-150/day
3. **Weekly rental**: €200-500/week
4. **Monthly rental**: €500-1500/month
5. **Setup/customization**: €100-500 one-time
6. **Training**: €100-200/hour

## Why Rent Instead of Buy?
- **High upfront cost**: Mac Studio M2 Ultra = €7000+
- **Rapidly evolving**: New models every 1-2 years
- **Occasional need**: Not everyone needs it daily
- **Try before buy**: Test workflows before committing
- **Burst capacity**: Scale up for big projects

## Rental Tiers

### Tier 1: Creator (€500/month)
- Mac Studio M1 Max 32GB
- Pre-installed creative apps
- Video editing, photo processing
- Basic AI tools (Whisper, SD)

### Tier 2: Professional (€800/month)
- Mac Studio M1 Max 64GB
- Full Adobe Creative Cloud
- DaVinci Resolve Studio
- Advanced AI (local LLMs, MLX)

### Tier 3: Enterprise (€1200/month)
- Mac Studio M2 Ultra 128GB+
- Custom software stack
- Priority support
- Dedicated resources

## Target Customers
- **Freelance video editors**: Need processing power for projects
- **Startups**: Can't afford to buy equipment
- **Agencies**: Burst capacity for deadlines
- **AI researchers**: Need Apple Silicon for MLX work
- **iOS developers**: Need Mac for App Store

## Technical Setup
```
Mac Studios →
  Tailscale VPN (secure remote access) →
  Each user gets isolated account →
  Shared storage (NAS) →
  Monitoring (uptime, usage) →
  Automated backup →
  Easy reset between users
```

## Logistics
- **Physical location**: Studio/office with good internet
- **Remote access**: Screen sharing, SSH, file transfer
- **Handoff**: Clean wipe between rentals (or persistent for monthly)
- **Support**: Remote assistance included

## Software Included
Pre-configured, licensed, ready to use:
- **Video**: DaVinci Resolve, Final Cut Pro
- **Photo**: Lightroom, Affinity Photo
- **Audio**: Logic Pro, Audacity
- **AI**: Ollama, ComfyUI, Whisper
- **Dev**: Xcode, VS Code, Docker

## Pricing Math
- Mac Studio M1 Max cost: €3000
- Lifespan: 4-5 years
- Monthly break-even: €50-60
- Monthly rental: €500-800
- **Margin**: 8-15x

## Scaling
Start with 2-3 Mac Studios:
- Revenue potential: €1500-4500/month
- Scale based on demand
- Consider Mac Mini fleet for lighter workloads

## Startup Costs
- 3× Mac Studio M1 Max: €9000-12000
- Network/infrastructure: €500-1000
- Software licenses: €200-500/month
- Legal (rental contracts): €500-1000
- Marketing: €500-1000

## Competitive Advantage
- **Pre-configured**: Ready to work immediately
- **Local**: Lower latency than cloud
- **Apple Silicon**: MLX, Neural Engine
- **Support**: Human help available
