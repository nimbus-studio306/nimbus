### Multi-Instance OpenClaw Discussion
- Key question: how many OpenClaw instances can run in parallel?
- Measured REAL resource usage:
  - Gateway RSS: 611MB after 2 days
  - CPU: 0.7% average, but PEAK during agent turn: 19% total (38% single core)
  - Pattern: I/O bound — 95/100 samples show 0% CPU, short bursts for parsing
  - 13 threads per gateway process
- Conclusions:
  - Mac Mini M4 Max 64GB: 10-15 instances comfortably
  - Mac Mini M1 8GB: 3 instances max, API-only, restricted tools
  - Real bottleneck: tool usage (browser, ffmpeg), not gateway processes
  - Each instance needs: own port, config, workspace, API keys
  - Separate API keys per instance to avoid rate limits
