- Everything else inside the container
- /home/node/.local/
- /home/node/.config/ (unless mounted)
- /tmp/

**Fix:** Add to startup.sh: install gh CLI if missing. Add volume mount for ~/.config/gh for auth persistence.

## Morning Session (05:00-07:00 UTC) — OpenClaw Config & Gmail Setup

### OpenClaw Gateway Config on nimbus-vm

**Config file created:** `~/.openclaw/openclaw.json`

**Key format learnings:**
- `gateway.bind` = `"loopback"` (NOT host:port format)
- `gateway.port` = `18789` (separate field)
- Auth profiles use nested structure: `auth.profiles.google:default.mode: "api_key"`
- No "providers" key - use `auth.profiles` directly

**Final working config structure:**
```json
{
  "gateway": {
    "bind": "loopback",
    "port": 18789
  },
  "auth": {
    "profiles": {
      "google:default": {
        "mode": "api_key",
        "api_key": "$GEMINI_API_KEY"
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "google/gemini-2.0-flash"
      }
    }
  }
}
```
