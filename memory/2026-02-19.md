### TTS Duplicate Voice Message Fix (IMPORTANT)
**Problem**: Voice messages were being sent twice.

**Root cause**: 
- `tts` tool auto-delivers the audio when called
- I was ALSO outputting the MEDIA line, causing second delivery

**Correct pattern documented in TOOLS.md:**
1. Call `tts({ text: "..." })` → auto-delivers
2. Do NOT output MEDIA line afterward
3. Respond with `NO_REPLY` or text-only acknowledgment

**Three TTS mechanisms:**
1. `tts` tool — explicit call, auto-delivers
2. `[[tts:text]]...[[/tts:text]]` — inline tags for mixed responses
3. `[[tts]]` — simple tag

Updated TOOLS.md on both VM and Mac Studio with these rules.
