## Technical Defaults

### Camera Settings
- **Format:** RAW (for editing flexibility)
- **Delivery:** High-res JPEG + optional RAW

### Editing Style
- Natural, clean edits
- Color correction, exposure balancing
- Light retouching (not heavy Photoshop)

### File Delivery
- **Preview:** Adobe Lightroom web gallery
- **Download:** Dropbox folder
- **Naming:** Project name + sequential number

## Pre-Shoot Checklist

### Client Communication
- [ ] Confirm number of edited images needed
- [ ] Discuss visual style / references
- [ ] Confirm location and timing
- [ ] Discuss wardrobe if relevant

### Logistics
- [ ] Scout location if new
- [ ] Check lighting conditions (golden hour, indoor/outdoor)
- [ ] Backup gear + cards

## Payment Terms

- **Advance:** 50% before shoot
- **Balance:** 50% after delivery
- Same as videography

## Delivery Workflow

1. **Ingest** — Backup all cards immediately
2. **Cull** — Select best shots
3. **Edit** — Color correct, light retouch
4. **Export** — High-res JPEG
5. **Deliver** — Lightroom preview + Dropbox download
6. **Invoice** — Send after delivery confirmed

## Common Client Questions

### "Can I get all the photos, not just edited ones?"
Discuss case by case. Usually no — curation is part of the service.

### "Can I use these for my album cover?"
Check licensing. Basic = no commercial use. Merchandising license = €250.

### "How long until I get the photos?"
- Preview gallery: 2-5 days
- Final delivery: 1-2 weeks
- Rush: Possible with surcharge
