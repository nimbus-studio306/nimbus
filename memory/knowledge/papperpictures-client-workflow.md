## Communication Principles

1. **Don't be overly enthusiastic** — Professional, not desperate. Zsolt is selective about clients.
2. **Evaluate the client** — Through communication, see how they respond, if they accept terms.
3. **Always require advance payment** — This is a standard term.
4. **Get details before confirming** — Location, event type, what they have in mind, deadlines.
5. **Post-production is extra** — Never assume editing is included.
6. **Check calendar/availability** — Don't confirm without verifying.

## Booking Workflow

1. **Client inquiry** — Via website form (picks service, date, sends message)
2. **Initial response** — Professional, ask for details (location, event, vision)
3. **Evaluate fit** — Is this a client Zsolt wants to work with?
4. **Discuss requirements** — What exactly do they need? Post-production? Audio engineer?
5. **Quote** — Based on package + extras
6. **Terms** — Advance payment required, deadlines discussed
7. **Confirmation** — Calendar blocked, contract/agreement
8. **Delivery** — Raw footage or edited (depending on what's agreed)

## Key Points for Drafting Responses

❌ **DON'T:**
- Sound too eager/enthusiastic
- Assume editing is included
- Confirm availability without checking
- Commit to anything without discussing terms
- Send responses directly to clients (always via Zsolt)

✅ **DO:**
- Be professional and warm, not desperate
- Ask for details (venue, event type, vision)
- Mention that post-production is available separately
- Ask about deadlines if they need editing
- Request a call to discuss specifics
- Mention advance payment will be required
