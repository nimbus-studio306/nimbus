## Technical Defaults

### Recording Format
- **Resolution:** 4K (default) or 1080p (if client prefers smaller files)
- **Codec:** ProRes (master) — explain file sizes to client upfront
- **Color:** S-Log3 (flexible grading) or Rec.709 (ready-to-use)

### File Sizes (4K ProRes)
- ~180GB for 90-minute session
- Warn clients about file sizes before shoot
- Offer 1080p option if they can't handle large files

### Multi-Cam Setup
- Two Camera package = 2 angles, synced in post
- Live Recording = more angles, potentially live switching

## Pre-Shoot Checklist

### Client Communication
- [ ] Confirm deliverables in writing
- [ ] Explain technical options (4K/1080p, color profile)
- [ ] Warn about file sizes
- [ ] Get venue details
- [ ] Confirm call time (arrive 1-2h early)

### Logistics
- [ ] Scout location if complex (low light, projectors, installations)
- [ ] Confirm audio needs (ambient? vox pops? dedicated recording?)
- [ ] Travel + load-in time
- [ ] Backup gear

## Delivery Workflow

1. **Ingest** — Backup all cards immediately
2. **Sync** — Align multi-cam footage + audio
3. **Export** — Master files in client-requested format
4. **Deliver** — Dropbox/WeTransfer for raw, Lightroom for previews
5. **Invoice** — Send after delivery confirmed

## Common Client Questions

### "Can I have 1080p instead of 4K?"
Yes. Smaller files, easier to handle. Quality still excellent for web/social.

### "What about audio?"
- Ambient: Camera mic is fine
- Important dialogue/performance: Dedicated audio recorder recommended (+€400)
- Vox pops: Clip mic or handheld

### "How long until I get the files?"
- Raw footage: 1-3 days
- Basic edit: 1-2 weeks
- Full production: Depends on scope, quote separately

### "Can you edit it too?"
Yes, quoted separately. €450/day or €75/hour. Need to discuss vision, references, deliverable length.
