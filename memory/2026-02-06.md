## Lessons Learned (Hard Night)
1. **Don't assume — check actual data**: I kept making assumptions instead of reading my own docs/configs
2. **Lists vs prose**: Zsolt wants readable documents, not bullet point dumps
3. **Knowledge graph isn't working yet**: Don't claim it as a feature until it's queryable
4. **Transcription broke**: Empty audio files (0 bytes) causing garbage transcriptions after ~05:57
5. **Anthropic auth**: Max subscription → authenticate → get token → put in OpenClaw (not automatic)
6. **Google Cloud**: Console credential with API services (not AI Studio key)
