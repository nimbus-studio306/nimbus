**CORRECT PROCESS for reply drafts:**
1. ALWAYS fetch the thread first: `gog gmail thread get <id>`
2. Get the EXACT original subject
3. Use "Re: [exact original subject]" — don't invent subject lines
4. `gog gmail drafts create` REQUIRES `--subject`, so must provide correct one

**Threading insight:** Gmail threading works on In-Reply-To/References headers + subject matching. Wrong subject = breaks threading even if reply-to-message-id is set correctly.
